# Global Radio & Podcast Platform — Bun migration ⚡

This repository has been updated to use Bun as the package manager.

Quickstart

- Install Bun: https://bun.sh
- From repository root:
  - Run `bun install` or `./setup.sh` (the setup script also installs frontend/backend deps)

Useful commands

- `bun run install` — install backend and frontend deps (root `package.json` script)
- `bun run dev:backend` — start the backend development server
- `bun run dev:frontend` — start the frontend dev server
- `bun run dev` — run both dev servers
